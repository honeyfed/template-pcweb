# 基于node的纯前端vue框架

# 安装依赖

```bash
npm i
```

# 前端开发

```bash
npm run dev
```

# 前端构建发布

```bash
npm run build
```

# 启动web服务

```bash
npm start
```

# 目录结构

- app express服务
- front 前端代码
  - build 前端构建脚本
  - config 构建配置
  - src 源码
    - api api封装
    - assets 资源文件
    - components 通用组件
    - error 错误页面，404页面
    - router 路由设置
    - utils 工具函数
    如果需要添加页面在src下面添加目录即可
  -static 纯静态文件
